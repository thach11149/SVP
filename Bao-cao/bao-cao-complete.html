2.4.4. Mô tả từng bảng dữ liệu

2.4.4.1. Bảng users:

STT Thuộc tính Kiểu dữ liệu Ràng buộc Diễn giải
1 id UUID PRIMARY KEY Mã định danh duy nhất
2 email VARCHAR(255) UNIQUE, NOT NULL Email đăng nhập
3 role VARCHAR(50) NOT NULL Vai trò (admin/technician)
4 full_name VARCHAR(255) NOT NULL Họ và tên
5 created_at TIMESTAMP DEFAULT NOW() Thời gian tạo

2.4.4.2. Bảng customers:

STT Thuộc tính Kiểu dữ liệu Ràng buộc Diễn giải
1 id UUID PRIMARY KEY Mã định danh khách hàng
2 name VARCHAR(255) NOT NULL Tên khách hàng
3 address TEXT NOT NULL Địa chỉ
4 phone VARCHAR(20) Số điện thoại
5 email VARCHAR(255) Email liên hệ

2.4.4.3. Bảng jobs:

STT Thuộc tính Kiểu dữ liệu Ràng buộc Diễn giải
1 id UUID PRIMARY KEY Mã định danh công việc
2 customer_id UUID FOREIGN KEY Mã khách hàng
3 technician_id UUID FOREIGN KEY Mã kỹ thuật viên
4 title VARCHAR(255) NOT NULL Tiêu đề công việc
5 description TEXT Mô tả chi tiết
6 status VARCHAR(50) NOT NULL Trạng thái công việc
7 scheduled_date DATE NOT NULL Ngày lên lịch
